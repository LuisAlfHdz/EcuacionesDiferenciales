Solucionador ecuaciones diferenciales homogeneas de segundo orden.



  El presente sistema tiene como objetivo resolver ecuacions diferenciales homogeneas de segundo orden.
  Para iniciar se debe llenar el formulario con los coeficientes de la ec. dif. y presionar el botón que se encuentra ahí.
  Cuando se desee pasar al siguiente paso, se debe presionar el boton que aparece al momento de que el nuevo paso aparece.
  Este sistema no solo resuelve el ejercicio propuesto, si no que da una breve explicación de cómo se hace.

Ecuacion diferencial a resolver
Aquí se plantea la ecuación diferencial de segundo orden homogénea.





Se plantea la ecuación caracterísitica de la ecuación dada.
Esta se obtiene tomando y = emx
Se deriva n veces(En este caso n = 2). Estas derivadas, junto con la forma planteada, se sustituyen en la ecuacón diferencial. Entonces tenemos:

        a2m2emx + a1memx + a0emx = 0

Factorizando términos comunes, en nuestro caso se obtiene:

       emx[a2m2 + a1m + a0] = 0

Como emx ≠ 0 se plantea la ecuación caracteristica como:

        a2m2 + a1m + a0 = 0

Una vez planteada la ecuación caracterísitica, se le da el tratamiento "de siempre" para encontrar raíces en de una ecuación de segundo grado.
Se puede factorizar, completar el trinomio cuadrado perfecto, o utilizar la fórmula general.
En el caso de este sistema, se utiliza la fórmula general.

Es importante muy importante tomar en cuenta que si las raíces de la ecuación caracterísitca son imaginarias, se debe dar el siguiente tratamiento:

       α será igual a la parte real de la solución.
       Por otro lado β será la parte imaginaria.






Se obtienen las soluciones linealmente indepentiendes.
En el caso de que las raíces de la ecuación característica sean reales, las soluciones se plantean de la siguiente manera:

       yc1 = em1x        yc2 = em2x

Para nuestro caso, si las raíces son iguales, entonces las soluciones linealmente independientes serán:

       yc1 = em1x        yc2 = xem1x


Por otro lado, si las raíces son imaginarias, las soluciones se plantean de la siguiente forma:

       yc1 = eαxcos(βx)        yc2 = eαxsen(βx)

Por último, para obtener la solución general, las soluciones linealmente dependientes se multiplican por 2 constantes arbitrarias diferentes(Generalmente denotadas como C1 y C2) y de esta manera se suman.
